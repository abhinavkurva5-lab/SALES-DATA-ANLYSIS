# SALES-DATA-ANLYSIS
ANALYZED 9800 ROWS OF SALES DATA USING PIVOT TABLES,VLOOKUP AND CHARTS
# Sales Data Analysis Project

## Tools Used
- Google Sheets
- Pivot Tables
- VLOOKUP/IF Formulas
- Charts

## What I Did
- Analyzed 9,800 rows of Superstore sales data
- Built Pivot Tables for monthly, product and regional analysis
- Used VLOOKUP to cross-reference category data
- Created Line and Bar charts to visualize trends
- Built KPI Dashboard
